# 秀视频--后台管理系统



# 相关技术 
 <p>前端:h5+css+javascript+jquery+bootstrap+themeleaf+ H plus</p>
 <p>后端:springboot+ sping+springmvc+mybatis+pagehelper+mysql+redis</p>
 <p>组件:bootsrap-table,webUploader.</p>
 <p>项目部署: docker + linux</p>
